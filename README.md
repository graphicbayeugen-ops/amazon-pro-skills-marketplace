# Amazon Pro Skills - Claude Plugin Marketplace

67 Amazon seller skills (originally by Jay Margaliot, https://github.com/JayGPTPro/amazon-pro-skills) repackaged as a Claude plugin marketplace for use in Claude apps (Cowork) and Claude Code.

## Install (Claude / Cowork)
Customize -> Plugins -> Personal plugins -> + -> Add marketplace -> graphicbayeugen-ops/amazon-pro-skills-marketplace, then install the "amazon-pro-skills" plugin.

## Install (Claude Code)
/plugin marketplace add graphicbayeugen-ops/amazon-pro-skills-marketplace
/plugin install amazon-pro-skills@amazon-pro-skills
